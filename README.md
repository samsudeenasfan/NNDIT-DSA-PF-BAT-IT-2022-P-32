## How to Compile and Run
1. Open a terminal.
2. Navigate to the folder containing the `MAXELEMENT.java` file.
3. Compile the file using the command:  
   `javac MAXELEMENT.java`
4. Run the program using the command:  
   `java MAXELEMENT`
